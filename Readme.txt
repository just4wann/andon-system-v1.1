1. Klik "Code" lalu "Download ZIP"
2. Extract file ZIP yang sudah didownload, nanti akan muncul folder:
- resource
- script
3. Buka folder `resource`, dan install Node Setup
4. Buat folder di drive C dengan path C:/App/andon
5. Pindahkan seluruh isi folder `script` ke C:/App/andon
6. Buka terminal di C:/App/andon, lalu jalankan "npm install" pastikan koneksi internet aktif
7. Dari folder `resource`, pindahkan file andon-system-start.bat ke Desktop
8. Double click file andon-system-start.bat untuk menjalankan aplikasi
